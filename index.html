Portafolio

En esta sección, podrás explorar algunos de los proyectos en los que he estado activamente involucrado, abarcando su desarrollo, mejora y optimización.

Inventario Campuslands

Este sistema de gestión de inventario facilita el control exhaustivo de artículos, categorías, marcas y proveedores, así como su asignación a usuarios específicos. Permite una administración eficiente mediante operaciones clave como registro, búsqueda, edición y eliminación de datos, asegurando un manejo ágil y preciso del inventario.

Tecnologías empleadas: HTML, CSS, JavaScript y Bootstrap 5.

Inventario Garden

Sistema integral para la gestión de un almacén de jardinería, que incluye operaciones CRUD completas y un robusto proceso de autenticación de usuarios. Este sistema permite una administración eficiente de inventarios, clientes y transacciones, mejorando la operatividad y la seguridad de la información.

Tecnologías empleadas: HTML, CSS, JavaScript, Bootstrap 5, SpringBoot - JWT y MySQL.
